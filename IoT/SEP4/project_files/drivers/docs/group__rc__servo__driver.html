Detailed Description

Driver to control the two RC-Servos that can be connected to the VIA ARDUINO MEGA2560 Shield rev. 2.0.

The RC-Servos can be connected to J14 (Servo#0) and J13 (Servo#1) on the VIA ARDUINO MEGA2560 Shield rev. 2.0.
